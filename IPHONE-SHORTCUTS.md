# iPhone 写博客快捷指令指南

## 📱 方案一：基于 Telegram 的快捷指令（推荐）

利用您现有的 Telegram 同步脚本，最简单快捷！

### 快捷指令功能
- ✅ 快速写博客标题和内容
- ✅ 可选择添加图片
- ✅ 自动发送到 Telegram 频道
- ✅ GitHub Actions 自动同步到博客

---

## 🛠️ 快捷指令创建步骤

### 第一步：创建新快捷指令

1. 打开 iPhone 上的「快捷指令」App
2. 点击右上角「+」创建新快捷指令
3. 命名为「写博客」或「发布博客」

### 第二步：添加询问标题动作

1. 点击「添加操作」
2. 搜索「询问」
3. 选择「询问输入」
4. 设置：
   - 提示：「请输入博客标题」
   - 输入类型：文本

### 第三步：添加询问内容动作

1. 点击「添加操作」
2. 搜索「询问」
3. 选择「询问输入」
4. 设置：
   - 提示：「请输入博客内容」
   - 输入类型：文本
   - 勾选「允许换行」

### 第四步：添加选择是否添加图片

1. 点击「添加操作」
2. 搜索「从菜单中选择」
3. 添加选项：
   - 「添加图片」
   - 「不添加图片」

### 第五步：添加获取图片动作（条件）

1. 在「添加图片」分支下：
2. 点击「添加操作」
3. 搜索「选择照片」
4. 选择「选择照片」

### 第六步：组合消息内容

1. 点击「添加操作」
2. 搜索「文本」
3. 选择「文本」
4. 输入：
   ```
   【标题】
   [询问标题的结果]

   【内容】
   [询问内容的结果]
   ```

### 第七步：发送到 Telegram

#### 方法 A：通过 Telegram App 发送
1. 点击「添加操作」
2. 搜索「Telegram」
3. 选择「发送消息」
4. 设置：
   - 收件人：您的频道（@FreePeriodical）
   - 消息：[组合的文本]
   - 如果有图片：添加「照片」变量

#### 方法 B：通过 URL Scheme 发送（更可靠）
1. 点击「添加操作」
2. 搜索「打开 URL」
3. 输入 URL（需要替换为您的 Bot Token 和 Chat ID）：
   ```
   https://api.telegram.org/bot[您的BotToken]/sendMessage?chat_id=@FreePeriodical&text=[编码后的文本]
   ```

## 📱 方案二：直接提交到 GitHub（高级）

如果您想跳过 Telegram，直接提交到 GitHub。

### 需要准备
- GitHub Personal Access Token
- 仓库路径：`lionel518/imperial-blog`

### 快捷指令步骤

1. **询问标题**
2. **询问内容**
3. **询问标签（可选）**
4. **生成文件名**（使用当前日期时间）
5. **生成 Markdown 内容**（包含 Frontmatter）
6. **调用 GitHub API 创建文件**

---

## 🎨 快捷指令增强功能

### 添加标签选择
- 创建一个菜单，包含常用标签：
  - 「技术」
  - 「生活」
  - 「思考」
  - 「读书」
  - 「自定义」

### 添加日期选择
- 默认使用当前日期
- 可选：选择未来日期（定时发布）

### 添加草稿选项
- 选择「立即发布」或「保存为草稿」
- 草稿会设置 `draft: true`

## 📋 完整的 Markdown 格式

快捷指令生成的内容应该符合以下格式：

```markdown
---
title: "博客标题"
pubDatetime: 2026-03-25T21:00:00+08:00
description: "博客描述"
heroImage: "/assets/blog/xxx.jpg"
tags: ["技术", "思考"]
draft: false
---

博客内容在这里...
```
