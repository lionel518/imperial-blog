# 📱 iPhone 写博客超简单指南

## 🎯 最简单的方案：直接用 Telegram！

因为您已经有 Telegram 自动同步脚本了，**不需要复杂的快捷指令**！

---

## ✅ 三步写博客

### 第 1 步：打开 Telegram

找到您的频道：**@FreePeriodical**

### 第 2 步：发送消息

**格式要求：**
- 第一行：**博客标题**
- 第二行及以后：**博客内容**
- 可选：添加图片（会自动成为 heroImage）

**示例：**
```
今天的思考

今天我想分享一下关于人工智能的一些想法...
```

### 第 3 步：等待自动同步

- GitHub Actions 每 **3 小时**自动同步一次
- 或者您可以手动触发同步（见下文）

---

## 🚀 如何手动触发同步？

如果您不想等3小时，可以手动触发：

1. 访问 GitHub 仓库：https://github.com/lionel518/imperial-blog
2. 点击「Actions」标签
3. 选择「Telegram to Astro Sync」工作流
4. 点击「Run workflow」→ 「Run workflow」

---

## 📸 带图片的博客

**直接在 Telegram 中发送带图片的消息：**

1. 选择一张图片
2. 在图片描述中输入：
   - 第一行：标题
   - 第二行及以后：内容

**示例：**
```
春天的美景

今天去公园看到了美丽的花朵...
```

图片会自动成为博客的 heroImage！

---

## 💡 小技巧

### 1. 使用 Telegram 的「保存消息」功能
- 先在「保存消息」中写好草稿
- 满意后再转发到频道

### 2. 使用 Markdown 格式
Telegram 支持简单的 Markdown：
- `**粗体**` → **粗体**
- `*斜体*` → *斜体*
- `` `代码` `` → `代码`

### 3. 添加标签
在消息末尾添加标签，会自动同步：
```
#技术 #思考 #生活
```

## ⚙️ （可选）创建简单快捷指令

如果您想更快捷，可以创建一个简单的快捷指令：

### 快捷指令步骤：

1. 打开「快捷指令」App
2. 创建新快捷指令，命名为「写博客」
3. 添加操作：「打开 App」→ 选择「Telegram」
4. 完成！

**使用方法：**
- 点击快捷指令 → 自动打开 Telegram
- 在 Telegram 中写博客 → 发送到频道
